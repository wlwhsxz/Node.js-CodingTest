### 해설
`O, X를 구분하여 카운트 하는 문제`  
`이중 for문 활용`  
`count 와 sum 변수를 구분하여 덧셈을 진행`  

### 답안
```
let input = require('fs').readFileSync('/dev/stdin').toString().split('\n');

let num = Number(input[0]);

for (let i = 1; i <= num; i++) {
  let count = 0;
  let sum = 0;

  for (let j = 0; j < input[i].length; j++) {
    if (input[i][j] === "O") {
      count++;
    } else {
      count = 0;
    }
    
    sum += count;
  }
  
  console.log(sum);
}
```
